# Street Light Management System

A modern, scalable, and responsive **Street Light Management System** built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS**. The application provides a centralized platform for monitoring, managing, and maintaining street lighting infrastructure through role-based access and an intuitive dashboard.

---

## 🚀 Features

- 🔐 Secure Authentication
- 👥 Role-Based Access Control (RBAC)
- 📊 Interactive Dashboard
- 💡 Street Light Management
- 🗺️ Ward & Zone Management
- 📍 Pole Management
- 🛠 Complaint Management
- 🔔 Notification System
- 📈 Analytics & Reports
- 🌙 Dark Mode Support
- 📱 Fully Responsive Design

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Marsalaan5/Street_Light_Nextjs.git
```

Go to the project directory

```bash
cd Street_Light_Nextjs
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
